Üçgensel Sayıların Kalan Döngüleri

Bu çalışma, UAMAiİ.pdf dosyasında detaylandırılan sistemin mantığını ve düzenini açıklamaktadır.

Üçgensel Sayılar Nedir?

Üçgensel sayılar, T(n) = n(n+1)/2 formülü ile hesaplanır.
Örneğin:

  • T(0) = 0
  • T(1) = 1
  • ...
  • T(15) = 120

Sırasıyla üçgensel sayılar dizisi şöyledir:
(1, 3, 6, 10, 15, 21, 28, 36, 45, 55, 66, 78, 91, 105, 120, ...)

Modüler Aritmetik ile İnceleme

T(n) dizisinin farklı modülo (m) değerlerine göre kalanlarını (T(n) mod m) incelediğimizde belirli döngülerle karşılaşırız.

  • m = 2 için: (1, 1, 0, 0, 1, 1, 0, 0, ...)
  • m = 3 için: (1, 0, 0, 1, 0, 0, ...)
  • m = 4 için: (1, 3, 2, 2, 3, 1, 0, 0, ...)
  • m = 5 için: (1, 3, 1, 0, 0, 1, 3, 1, 0, 0, ...)
  • m = 6 için: (1, 3, 0, 4, 3, 3, 4, 0, 3, 1, 0, 0, ...)
  • m = 7 için: (1, 3, 6, 3, 1, 0, 0, 1, 3, 6, 3, 1, 0, 0, ...)

Bu dizilerde şu kurallar gözlemlenmiştir:

  • m tek ise: T(m-1) ≡ T(m) ≡ 0 (mod m)
  • m çift ise: T(2m-1) ≡ T(2m) ≡ 0 (mod m)

Örnek döngüler:
Mod 8: (1, 3, 6, 2, 7, 5, 4, 4, 5, 7, 2, 6, 3, 1, 0, 0)
Mod 9: (1, 3, 6, 1, 6, 3, 1, 0, 0)
Mod 10: (1, 3, 6, 0, 5, 1, 8, 6, 5, 5, 6, 8, 1, 5, 0, 6, 3, 1, 0, 0)

Bu sonuçlardan, birler basamağı (2, 4, 7, 9) olan hiçbir üçgensel sayı olmadığını görebiliriz.

Döngü Toplamları İncelemeleri

Her bir mod değeri için oluşan döngüdeki sayıların toplamını inceleyelim:

  • Mod 2: Döngü (1, 1, 0, 0). Toplam = 2.
  • Mod 3: Döngü (1, 0, 0). Toplam = 1.
  • Mod 4: Döngü (1, 3, 2, 2, 3, 1, 0, 0). Toplam = 12.
  • Mod 5: Toplam = 5.
  • Mod 6: Toplam = 22.

Not: Mod 4 için, ilk 8 (çift olduğu için 2*4) üçgensel sayının mod 4'teki değerlerini topluyoruz.
Mod 5 için, ilk 5 (tek olduğu için 1*5) üçgensel sayının mod 5'teki değerlerini topluyoruz.

Bu toplamları, 1. sıraya 0 koyarak sıraladığımızda şu dizi ortaya çıkar:
(0, 2, 1, 12, 5, 22, 14, 56, 21, 70, 44, 116, 52, 154, 65, ...)

Oranlar ve Bağlantılar

Döngü toplamları ile mod sayısı (m) arasında bir bağlantı olup olmadığını anlamak için (Döngü Toplamı) / (Mod Sayısı) oranına bakabiliriz.

Örnekler:

  • Mod 2 için toplam 2 idi. Oran: 2/2 = 1.
  • Mod 3 için toplam 1 idi. Oran: 1/3 ≈ 0.333.

Mod 2'den başlayarak ilk 15 sayı için oranlar şöyledir:
(1, 0.333, 3, 1, 3.666, 2, 7, 2.333, 7, 4, 9.666, 11, 4.333)

3'ün Katları

Sadece 3'ün katlarındaki oranları sıralarsak:

(0.333, 3.666, 2.333, 9.666, 4.333, 13.666, ...)

Burada ilginç bir desen vardır:
- 3'ün katı olan çift sayıların oranında virgülden sonra devreden kısım 6'dır.
- 3'ün katı olan tek sayıların oranında virgülden sonra devreden kısım 3'tür.

Yakınsama Durumu

Oranlar incelendiğinde şu yakınsamalar görülmektedir:

  • Çift bir sayının döngü toplamının, bir önceki sayının döngü toplamına oranı 2'ye,
  • Tek bir sayının döngü toplamının, bir önceki sayının döngü toplamına oranı 0.5'e,
  • Çift sayının döngü toplamının, bir önceki çift sayının döngü toplamına oranı 1'e,
  • Tek sayının döngü toplamının, bir önceki tek sayının döngü toplamına oranı 1'e yakınsar.

Merkez Sayı ve İyonlaşma Enerjisi Benzerliği

Tek sayılarda "merkez sayı"yı bulmak için kullanılan formül:

(m² - 1) / 8 (mod m)

İlk 50 sayı için merkez sayı dizisi:

(0, 1, 1, 2, 3, 3, 6, 4, 1, 5, 4, 6, 8, 7, 13, 8, 2, 9, 7, 10, 13, 11, 20, 12, 3, 13, 10, 14, 18, 15, ...)

Resmi İndir

Asal Sayılar ve Oranlar

Döngü toplamlarının asal sayılara (p) oranını incelediğimizde:
Mod 5 için toplam 5, oran 5/5 = 1.
Mod 7 için toplam 14, oran 14/7 = 2.

Döngü toplamının, asal sayının karesine oranının 1/2'ye yaklaştığı gözlemlenebilir.
